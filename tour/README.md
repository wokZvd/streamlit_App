# 🐶 PawTrip Korea

## 반려견과 함께 떠나는 국내 여행

한국관광공사 TourAPI와 AI 일정 추천 기능을 활용한  
반려견 동반 여행 플랫폼입니다.

반려견과 함께 갈 수 있는:

- 🏕️ 관광지
- 🏨 숙소
- 🍽️ 음식점
- ☕ 카페
- 🌲 자연 여행지

를 검색하고 AI가 맞춤 여행 일정을 추천합니다.

---

# ✨ 주요 기능

## 🗺️ 여행지 검색

한국관광공사 국문 관광정보 서비스(TourAPI)를 활용합니다.

지원 기능:

- 지역 검색
- 키워드 검색
- 관광지 조회
- 이미지 표시
- 주소 표시
- 전화번호 표시
- 반려견 동반 필터


---

## 🐾 지도 기능

Folium 기반 인터랙티브 지도 제공

기능:

- 검색 결과 Marker 표시
- 위치 확인
- 장소 정보 Popup


---

## 🤖 AI 여행 일정 생성

사용자 입력:

- 여행 지역
- 여행 기간
- 반려견 크기
- 여행 스타일

선택:

- 🌲 자연
- 🏕️ 캠핑
- ☕ 카페
- 🐾 힐링
- 🏄 액티비티
